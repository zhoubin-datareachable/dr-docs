前端项目规范

该文档是前端项目规范文档，请仔细阅读相关文档条例。

项目 markDown 规范列表

  • 标题

    项目规范文档的标题请使用 Markdown level 1 字体

    可选项：项目的标题应该包含一个集中说明的图像或者徽章，请引用Shields

  • 描述

    项目的描述标题请使用 Markdown level 1 字体

    项目描述的内容请使用 Markdown 普通文本字体

  • 项目结构

    项目结构标题请使用 Markdown level 1 字体

    Subsections:

    子标题请使用 Markdonw Level 2 字体.

    1. 项目结构代码
      项目的代码结构源码可以通过 vs code extentsion:directory-tree generator自动生成。
    2. 根目录描述
      该部分应该包含了更目录中每个重要文件的详细描述。
    3. Src 目录描述
      该部分应该包含了在 src 目录下一些重要文件/文件夹的详细描述。
    4. 其他目录描述
      其他目录的描述包含了test， *.config 等文件/文件夹的详细描述。
