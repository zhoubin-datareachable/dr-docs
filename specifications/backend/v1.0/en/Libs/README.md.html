Required Libraries (NPM Packages) When Relevant

This document lists the required library specifications that must be followed to complete a certain task. Not all of them are required in each project, but when relevant, usage of these libs should be enforced.

Note: Advices on this list are always welcome. If you notice any issues/bugs in the current libs or you have better ones to introduce, don't hesitate to speak out.

⚠️ Important Notice

The versions in the following lists might be changed to the latest versions in actual projects. The versions in this document is not updated to the latest ones.
