Redis Driver

The Redis application in development cluster is deployed in cluster mode.

USE ioredis NodeJS driver to connect to the Redis cluster, because it is full-featured though sacrifices on performance.

DO NOT USE node_redis, because it does NOT support Redis cluster mode.

Refer to database info for connection to the Redis cluster. And refer to the Redis official documentation to learn how to use Redis cluster (in sharding mode).

:bell: Notes

  1. We use redis cluster mode. Don't blindly use {} in the redis keys. Redis cluster use the contents inside {} to decided the slot allocation of the key. Unless you do want to store a group of keys on the same node, don't use the curly braces for other purposes.
