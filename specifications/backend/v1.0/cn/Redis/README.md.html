Redis 驱动

开发用Kubernetes集群中部署的Redis使用了分片集群模式。

使用 ioredis NodeJS 驱动连接Redis集群，因为该驱动功能齐全，虽然牺牲了性能。

禁止使用 node_redis驱动，因为其不支持Redis的集群模式。

请参考数据库信息来了解如何连接开发集群中的Redis集群。参考Redis官方文档学习如何使用分片模式下的Redis集群。

:bell: 备注

  1. 我们实际使用Redis集群模式部署。不要在Redis的键中滥用{}。Redis集群根据键中{}之间的内容确定该键分配的slot。除非你确实需要将一组键固定到同一个Redis节点上，否则，不要为了其他目的而使用{}。
