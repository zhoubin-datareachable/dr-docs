规定使用的库（在需要时）

本文档列出了要求使用的库列表来完成某个功能。并不需要在每一个项目中使用这里所有的库，但是，当需要完成相关的功能时，这些推荐库是必须被使用的。

**注意:**我们欢迎任何对于此列表的建议。如果你发现此列表中的库存在bug或其他问题，或者你有更好的替代库推荐，请第一时间提出。

⚠️ 重要通告

下面列表中的版本号在实际的项目中会更新。本文档中的版本号可能无法实时更新到相关依赖的最新版本。
