Usage

Local Environment Requirement

This project requires React>=16.13.0,NodeJS>=12.0.0 and Docker>=2.4.0.

The following procedure assumes the nodejs, npm, and docker have already been installed and are accessible from the commandline. For installation of node and npm, see install node and npm. For installation of docker, see install docker.

Run the App

Run the following command to install the packages:

npm install

Run the demo:

npm run start

Build a Docker Image

Build a deployment-ready docker image:

npm run docker:build
