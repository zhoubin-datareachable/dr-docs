使用方法

本地环境配置

本项目需要: React>=16.13.0; NodeJS>=12.0.0; Docker>=2.4.0。

下面假定已经在本地安装了NodeJS、NPM包管理器、和Docker，而且可以直接通过命令行调用。关于如何安装NodeJS和NPM，请参阅：安装NodeJS和NPM。关于如何安装Docker，请参阅：安装Docker。

运行本应用

首先运行如下命令来安装必须的NPM包：

npm install

运行程序示例：

npm run start

构建一个Docker镜像

运行如下命令构建一个Docker镜像：

npm run docker:build
