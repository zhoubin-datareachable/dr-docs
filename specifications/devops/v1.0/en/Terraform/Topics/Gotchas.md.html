Gotchas

1. Version Pinning

You should pin all of your Terraform modules to a specific version of Terraform, because each major release of Terraform is backward incompatible: e.g., upgrading from 0.11.x to 0.12.x requires many code changes, so you donâ€™t want to do it by accident.

terraform {
  # Require any 0.14.x version of Terraform
  required_version = ">= 0.14, < 0.15"
}

You should also pin all of your provider versions:

provider "aws" {
  region = "ap-southeast-2"
  # Allow any 3.x version of the AWS provider
  version = "~> 3.0"
}

2. Plan Before Apply

Always run terraform plan (or terragrunt plan) and check the outputs before calling terraform apply (or terragrunt apply).
